# Usage
## Requirements
* virtualenv
```sh
$ pip install virtualenv
$ . venv/bin/activate
(venv)
```
* Flask
```sh
$ pip install Flask
```
* Peewee
```sh
$ pip install peewee
```

## Launch
```sh
$ export FLASK_APP=app.py
$ flask run
```

そのうちサーバ起動スクリプトも書く
